Online Examination System

Project Overview

Now-a-days, Online Examination System has become popular for competitive examinations because of its unique features such as auto-evaluation, speed and accuracy. Moreover, it also helps environments by reducing the use of paper. In such a system, students are asked to select answers from multiple options given for a single question. Likewise, there are several questions which appear in the students’ systems. The questions and multiple options are saved in a database along with desired answers. Typically, a student can edit an answer after saving it, however, editing cannot be done after submitting the answer. Another user is also there – administrator. The administrator can create, modify and delete questions and accordingly, the question is updated in the system.


Role of Administrator:

  1. If an administrator wants to login, he will login using his login credentials.

  2. Login credentials-> Administrator (Email Address, Password).

  3. Only administrator can CREATE, MODIFY or DELETE questions.

Role of Student:

  1. This app is mainly used by students.

  2. If an student wants to login, he will login using his login credentials.


    1. If a student is new to this app he has to Sign up
      Sign up Student Information (Name, Roll No, Email Address, Contact Number, Password).


    2. If a student is an existing user he has to Sign in
      Sign in Student Information (Email Address, Password).

  3. After the student login he/she will be able to see the exams available to attempt.

  4. As it is a timed exam, the exam will be automatically submitted after the time has completed.

Analytics you can view after exam:

  1. After submission the marks obtained by the student will be displayed.

  2. Student can view the results in DESCENDING order according to the marks obtained with Roll Number.

  3. He can view some additional exam analytics also.
